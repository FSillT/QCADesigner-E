Keyboard Shortcuts

The following QCADDesigner features can be accessed by keyboard shortcuts:
Ctrl+NNew File
Ctrl+OOpen File
Ctrl+SSave File
Ctrl+WClose File
Ctrl+QQuit QCADesigner
Ctrl+ZUndo
Ctrl+Shift+ZRedo
DeleteDelete Selection
WZoom In
QZoom Out
Ctrl+EZoom Extents
1Reset Zoom
Ctrl+LZoom Layer Extents
Ctrl+RRotate Selection 90°
Ctrl+IIncrement Cell Clocks
Ctrl+HHide All But The Selected Layer
Ctrl+MStart Simulation
Ctrl+TStop Simulation
F1Show Help
